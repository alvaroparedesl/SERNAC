Description

Gráfico de la serie

Usage

plotme(
  db,
  t_observado = "min",
  pos = 1,
  ascendente = TRUE,
  clase = "industria",
  metrics = NULL
)

Arguments

db

objeto retornado por 'computar_reclamos'-

t_observado

periodo observado

pos

posición del ranking a observar

ascendente

se muestren de menor a mayor?

clase

nombre de la clase que se desa visualziar. Debe corresponder al nombre asignado a 'byClase' en 'computar_reclamos'.

metrics

métricas a visualizar. Si no se pasa argumento, se usarán todas.
